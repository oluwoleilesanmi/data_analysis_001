
## Description
- This [Kaggle Dataset](https://www.kaggle.com/datasets/rtatman/digidb) contains information on digimon from **Digimon Digimon Story: Cyber Sleuth**, released for Playstation Vita in 2015 and Playstation 4 in 2016.
- Digimon, short for **digital monsters**, is a franchise that revolves around a core mechanic of capturing, caring for and training monsters and then engaging in combat with them. It’s similar to Pokémon.
